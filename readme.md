Keyboard Shortcuts
==================

- `1`: Re-center in the middle of the map.
- `2`: Scale back.
- `3`: Scale forward.
- `esc`, `m` or `right click`: Go back to the top level.


Font and Images
===============

Both the images and the font are from the game diablo3.
